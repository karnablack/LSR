# LSR · Creative Visualizer (frontend)

React + Vite dashboard for the **LSR Synaptic Response Center**.

This is the UI only. See the [root README](../README.md) for the full project,
architecture, and setup instructions.

```bash
npm install
npm run dev      # http://localhost:5173
```

The dashboard expects the backend on `http://127.0.0.1:8000`
(start it from [`../reasoning-agent`](../reasoning-agent)).
